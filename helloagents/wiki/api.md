# API 手册

## 概述
浏览器统一访问前端容器暴露的 HTTP 端口。所有 `/api/v1` 请求由前端 Nginx 反向代理到后端 Spring Boot 服务。

## 认证方式
后端使用 Bearer Token 认证；未认证接口如注册、登录、发送验证码可匿名访问。

---

## 接口列表

### 代理入口

#### [ANY] /api/v1/**
**描述:** 由前端容器转发到后端容器的业务接口入口。

**请求参数:**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| path | string | 是 | 后端实际接口路径 |

**响应:**
```json
{
  "code": 0,
  "data": {}
}
```

**错误码:**
| 错误码 | 说明 |
|--------|------|
| 502 | 前端代理无法连接后端 |

#### [GET] /actuator/health
**描述:** 透传后端健康检查接口，便于远程部署后确认服务状态。

**请求参数:**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 无 | - | 否 | - |

**响应:**
```json
{
  "status": "UP"
}
```

**错误码:**
| 错误码 | 说明 |
|--------|------|
| 503 | 后端未就绪 |
