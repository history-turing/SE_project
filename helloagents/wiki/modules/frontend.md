# frontend

## 目的
提供武大树洞前端页面，并在部署时通过 Nginx 对浏览器暴露统一入口。

## 模块概述
- **职责:** 构建 React 静态资源、承接浏览器访问、将 `/api/v1` 请求代理到后端
- **状态:** ✅稳定
- **最后更新:** 2026-04-09

## 规范

### 需求: 远程服务器一键部署
**模块:** frontend
前端镜像需在容器内完成构建，并通过 Nginx 提供单页应用与接口代理能力。

#### 场景: 浏览器访问远程服务器
用户访问服务器公开端口时，应直接获得前端页面，并能正常调用后端 API。
- 返回 `dist/` 构建产物
- 所有前端路由支持刷新回退到 `index.html`
- `/api/v1` 请求通过反向代理转发到 backend 容器

## API接口
### [ANY] /api/v1/**
**描述:** 通过 Nginx 转发到后端
**输入:** 浏览器原始请求
**输出:** 后端 JSON 响应

## 数据模型
### VITE_API_BASE_URL
| 字段 | 类型 | 说明 |
|------|------|------|
| VITE_API_BASE_URL | string | 前端构建时注入的 API 基础路径，默认 `/api/v1` |
