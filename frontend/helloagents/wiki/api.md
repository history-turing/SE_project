# API 手册

## 概述
当前版本为纯前端演示站点，不依赖外部 API。所有页面内容与交互均由本地种子数据和浏览器本地存储提供。

## 认证方式
当前版本未接入登录与认证机制。

---

## 接口列表

### 前端本地交互

#### [LOCAL] composePost
**描述:** 创建新的树洞帖子，并同步到首页、个人页及可选的校友圈。

#### [LOCAL] toggleLike / toggleSave
**描述:** 切换帖子点赞与收藏状态，并在本地数据中回写统计值。

#### [LOCAL] sendMessage
**描述:** 向当前会话追加一条新消息并更新最后消息摘要。
